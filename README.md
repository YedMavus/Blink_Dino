# Blink_Dino

Contains code to two games:
1. Eye blink dino - control the dinosaur on no internet chrome page, using eye blinks
2. Control flappy bird on screen using webcam feed with the help of pinching movements using fingers.

Create python environment using default conda environment.
Install uv using pip install uv if not already done.
`uv venv` and then install dependencies with `uv pip install -r requirements.txt`

To run, clone repo, go to repo directory and run using
```python eye_blink.py --shape-predictor shape_predictor_68_face_landmarks```

For 2, run using `python hand_gesture.py`
